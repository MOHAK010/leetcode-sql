# Histogram of Tweets

**Difficulty:** Easy  
**Platform:** DataLemur

## Approach

First, calculate the total number of tweets made by each user in 2022.

Then, group users by their tweet count to create tweet buckets and count how many users belong to each bucket.

## Solutions

- `solution-subquery.sql` — Solution using a subquery
- `solution-cte.sql` — Solution using a Common Table Expression (CTE)
